# WebITpj
**Realtime collaborative whiteboard web app using Node Js React Js**

Collaborative board web applications are becoming increasingly popular in todayâ€™s world, where remote work is becoming more common. These applications allow users to collaborate on a virtual whiteboard in real-time, making it easier to share ideas and work together on projects.

**How to run:**
1. Clone to project to your local machine.
1. Open the project and navigate to the front end folder with package.json
1. Call the command "yarn dev"
1. The server will start on the value of the port
1. Navigate to http://localhost:(port)
